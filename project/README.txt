---INSTALLATION STEPS---

1. Install xampp application.
2. Copy PROJECTREPOSITORY folder then         paste it inside on the xampp\htdocs folder.
3. Navigate http:/localhost/phpmyadmin on your         internet browser.
4. Create new database, and name it "project"
5. Click project database and select import.
6. Select project.sql in the             PROJECTREPOSITORY folder, and click "Go".
7. Navigate http:/localhost/projectrepository
